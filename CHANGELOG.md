# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-04

### Added
- Initial MVP release
- Global ranking of top 10 QA professionals
- Profile cards with LinkedIn follower counts
- Position change indicators (↑ ↓ ─)
- Responsive design (mobile, tablet, desktop)
- Desktop table view and mobile card view
- TypeScript type safety throughout
- Tailwind CSS for styling
- JSON-based data storage
- Documentation for deployment
- LinkedIn data collection strategy guide
- Docker support
- Example environment variables

### Features
- Ranking calculation with historical comparison
- Automatic position change tracking
- Clean, modern UI with Lucide icons
- SEO-optimized with Next.js metadata
- Static generation for fast performance
- Fallback avatars for missing profile pictures

### Documentation
- Comprehensive README
- Deployment guide (VPS, Vercel, Docker, Netlify)
- LinkedIn data collection strategies
- Project structure documentation
- Contributing guidelines
