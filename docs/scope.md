# Escopo

## O que entra no ranking
- Profissionais de QA com perfil publico no LinkedIn.
- Seguidores no LinkedIn como metrica principal (MVP).
- Dados salvos em snapshots historicos no JSON.

## O que fica fora
- Metricas privadas ou nao publicas.
- Comparacao de qualidade tecnica.
- Conteudo patrocinado, ads ou paywall.

## Regras de negocio (alto nivel)
- Ordenar por followers (decrescente).
- Exibir mudanca de posicao por snapshot.
- Cada perfil deve ter id unico e URL valida.

## Fluxos principais do usuario
- Acessar ranking global.
- Ver perfil e link para LinkedIn.
- Interpretar mudanca de posicao (subiu, caiu, manteve).

## Quando houver duvida
Este documento define o que faz parte do produto e o que nao faz.
